VelocityNvim

Native vim.pack Neovim Distribution

Du wirst automatisch zum GitHub-Repository weitergeleitet...

Falls nicht, klicke hier: VelocityNvim auf GitHub

Features

  • Native vim.pack Integration - keine externen Plugin-Manager
  • Moderne LSP-Konfiguration mit vim.lsp.config
  • Treesitter für Syntax-Highlighting
  • Telescope für Fuzzy Finding
  • Performance-optimiert für WezTerm
  • Umfangreiche Dokumentation

Schnellstart

git clone https://github.com/Maik-0000FF/VelocityNvim.git ~/.config/VelocityNvim
NVIM_APPNAME=VelocityNvim nvim
